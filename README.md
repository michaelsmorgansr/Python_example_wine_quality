🍷 Wine Quality Prediction with Machine Learning

Welcome to my data science project using the **UCI Wine Quality dataset**!  
In this project, I predict wine quality based on physicochemical properties using **regression and classification models**.

---

## 📊 Project Overview

This project walks through a complete machine learning workflow:

- Data loading and exploration
- Data preprocessing and feature engineering
- Regression modeling to predict **wine quality scores (0–10)**
- Classification modeling to predict **“good” wine (quality ≥ 7)**
- Model evaluation and interpretation
- Visualizations and feature importance analysis

**Goal:**  
Use measurable chemical properties of wine to understand and predict quality.

---

## 🧠 Skills Demonstrated

- Data analysis and cleaning with **Pandas**
- Numerical computing with **NumPy**
- Machine learning with **scikit-learn**
- Data visualization with **Matplotlib**
- Regression and classification modeling
- Model evaluation (RMSE, R², ROC-AUC, Precision/Recall)
- Feature importance and model interpretation
- Reproducible project structure (GitHub-ready)

---

## 📁 Dataset

The dataset used in this project comes from the **UCI Machine Learning Repository**:

**Wine Quality (Vinho Verde) Dataset**

- Two variants: **Red wine** and **White wine**
- Each row represents one wine sample
- Target variable: `quality` (integer score from 0 to 10)

### Key Features Include:
- Fixed acidity
- Volatile acidity
- Citric acid
- Residual sugar
- Chlorides
- Free sulfur dioxide
- Total sulfur dioxide
- Density
- pH
- Sulphates
- Alcohol
- Wine type (red / white)

📥 The dataset is downloaded automatically by the notebook or script.
